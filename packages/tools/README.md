## 使用步骤

### 1. 安装依赖包

```
npm i @nicecode/tools --save

// or yarn add @nicecode/tools
```

### 2. 在项目中引用

本文以 common 通用模块为例
```
import { common } from '@nicecode/tools'

common.generateUUID()
```
