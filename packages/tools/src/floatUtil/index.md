## floatUtil

数字运算方法合集

### 加法

```js
import { floatUtil } from '@nicecode/tools';

let count = floatUtil.add(12, 33);
```

### API

| 方法     | 说明 | 类型     | 默认值 |
| -------- | ---- | -------- | ------ |
| sub      | 减法 | function | -      |
| add      | 加法 | function | -      |
| devide   | 除法 | function | -      |
| multiply | 乘法 | function | -      |
